# Descrição do Dataset

## Fonte dos Dados
O dataset utilizado é o **Corporate Employee Performance and Productivity Dataset**,
uma base pública contendo informações relacionadas a funcionários, cargos,
desempenho e comportamento organizacional.

## Características Gerais
- Total de registros: ~100.000
- Formato: CSV
- Estrutura: tabular
- Granularidade: 1 linha por funcionário
- Domínio: Recursos Humanos / People Analytics

## Principais Atributos
O dataset contém informações como:
- Identificação do funcionário
- Departamento e cargo
- Dados demográficos
- Data de contratação
- Métricas de desempenho e produtividade
- Indicadores de treinamento, promoções e satisfação
- Indicador de desligamento (Resigned)

## Observação Importante
Embora seja classificado como dado RAW para fins de ingestão na plataforma,
este dataset já apresenta sinais de tratamento prévio pelo fornecedor,
como nomes de colunas padronizados e métricas calculadas.

Essa característica será considerada nas etapas de organização e governança.

Link de Referência: https://www.opendatabay.com/data/ai-ml/3c34a52d-b417-4ba9-9c4a-d1a634c6a167

## Cobertura Temporal
Os dados cobrem um período aproximado de **setembro de 2014 a setembro de 2024**,
com informações de funcionários entre 22 e 60 anos.

## Casos de Uso Comuns
Este dataset é amplamente utilizado para:
- Análise de desempenho e produtividade
- Avaliação de satisfação e retenção
- Estudos de churn de funcionários
- Análises estratégicas de RH

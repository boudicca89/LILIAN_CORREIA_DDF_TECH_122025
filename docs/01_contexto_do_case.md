# Contexto do Case

Este projeto simula a atuação de um Profissional de Dados da Dadosfera
em um cliente do setor de e-commerce que está iniciando a construção
de sua Plataforma de Dados.

O objetivo do cliente é centralizar dados de diferentes áreas para
viabilizar análises descritivas e prescritivas com maior agilidade,
menor custo e maior confiabilidade.

Neste contexto, o foco deste case é a área de Pessoas (RH),
utilizando dados de desempenho e produtividade de funcionários
para apoiar análises de performance, eficiência operacional
e retenção de talentos.
